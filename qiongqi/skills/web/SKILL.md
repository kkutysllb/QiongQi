# Web Skill

Use the web tools (`web_search`, `web_fetch`) when the answer depends on information outside the local workspace — recent releases, library docs, current events.

- Prefer a targeted search, then fetch the most relevant page.
- Quote or cite the source URL for any factual claim.
- Fall back to local knowledge only when web tools are unavailable or the query is self-contained.
